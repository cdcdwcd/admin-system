若依后台管理系统 - 实训项目
基于若依(RuoYi-Vue)框架开发的后台管理系统，用于课程实训与学习展示

项目介绍
本项目是一套通用型后台管理系统，包含用户管理、角色权限、菜单管理、部门管理、字典管理、定时任务等核心功能，采用前后端分离架构。

技术栈
后端：Java 8、Spring Boot、Spring Security、MyBatis、MySQL
前端：Vue 3、Element UI、Axios

功能模块：用户管理、角色管理、菜单管理、部门管理、字典管理、定时任务、系统日志、代码生成

部署说明
本项目曾在 Sealos Cloud 云平台部署
当前云服务已到期下线，可本地运行
运行前请修改数据库配置文件

项目结构
zzyl-admin 管理后台入口
zzyl-common 公共模块
zzyl-framework 框架核心
zzyl-system 系统业务模块
zzyl-ui 前端页面

注意事项
1.已隐藏数据库账号、密码、地址、端口等敏感信息
2.仅保留核心源码，无本地缓存与依赖文件
